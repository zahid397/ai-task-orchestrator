About AI Task Orchestrator

This application demonstrates AI task orchestration using IBM watsonx for the IBM Dev Day AI Demystified Hackathon.

Features:

  • Planner Agent: Breaks down complex tasks into manageable steps
  • Validator Agent: Validates task feasibility and identifies risks
  • Advisor Agent: Provides expert recommendations and best practices
  • Real-time Orchestration: Coordinates multiple AI agents
